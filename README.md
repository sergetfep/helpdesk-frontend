# HelpDesk Frontend

Frontend-часть для задания **HelpDesk**.

Бейджик сборки: [![CI](https://github.com/sergetfep/helpdesk-frontend/actions/workflows/web.yml/badge.svg?branch=main)](https://github.com/sergetfep/helpdesk-frontend/actions/workflows/web.yml)

## Деплой

- Frontend опубликован на GitHub Pages: [HelpDesk](https://sergetfep.github.io/helpdesk-frontend/)
- Backend развёрнут на [Railway]([https://railway.com/](https://chat-backend-production-3fb4.up.railway.app)). Исходники [здесь](https://github.com/sergetfep/helpdesk-backend)

## Запуск

```bash
npm i
npm start
```

Изначально Frontend отправлял запросы на `http://localhost:7070`.

## Что реализовано

- загрузка списка тикетов;
- просмотр подробного описания по клику на тикет;
- добавление нового тикета;
- редактирование тикета;
- удаление тикета;
- переключение статуса выполнения.
